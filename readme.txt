This plugin allows you to add tags to pages and display them in a tag cloud widget.

== Description ==

With this plugin you can add tags to pages like you can to posts by default. The plugin also adds a new widget to display the page tags.

== Installation ==

1. Upload `page-tag-cloud` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= Does this plugin support a tag cloud on my homepage which displays my latest posts ? =

It does. Select the 'Show on home' option of tags you would like to show on your latest posts homepage.
